# Anime Recommendation Telegram Bot

A Telegram bot that recommends anime based on genre, ratings, episodes, and status.

## Features
- Genre-based recommendations
- Episodes count
- MyAnimeList rating (approx)
- Ongoing/completed status
- Random suggestions tab

## Setup

### 1. Install dependencies
pip install -r requirements.txt

### 2. Set your bot token
export BOT_TOKEN=your_token_here

### 3. Run the bot

python anime_bot.py
